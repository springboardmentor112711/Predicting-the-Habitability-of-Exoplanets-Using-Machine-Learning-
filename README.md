The NASA Exoplanet Habitability Prediction System is a data-driven web application designed to analyze exoplanetary data and predict whether a planet has the potential to be habitable or non-habitable. This project bridges data science, machine learning, and web development, providing an end-to-end solution that transforms raw astronomical data into meaningful insights through prediction and visualization.

With the increasing discovery of exoplanets by space agencies, identifying potentially habitable planets has become a critical research area. This project aims to simplify that process by using historical planetary data and applying predictive modeling techniques to classify planets based on their habitability indicators.

The system is built with a Flask-based backend, a clean and responsive frontend using HTML and CSS, and an interactive dashboard that allows users to explore planetary features and predictions in a visually engaging way.
